# 📦 PyPack – Installateur Python Minimaliste & Portable

PyPack est un outil léger permettant d’installer des modules Python, gérer des environnements simples et utiliser une interface GUI ou Terminal selon les besoins.  
Compatible Windows 64-bit et Linux 64-bit.
