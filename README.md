# 🐍 PyPack — Gestionnaire de paquets Python moderne et intuitif

PyPack est une application desktop légère et rapide permettant de rechercher, installer, mettre à jour et gérer les paquets Python sans jamais toucher au terminal.  
Pensé pour les développeurs comme pour les débutants, PyPack combine **simplicité**, **performance**, et **accessibilité internationale**.

---

## ✨ Fonctionnalités principales

- 🔍 **Recherche intelligente**  
  Fuzzy matching, suggestions instantanées, résultats robustes même hors‑ligne.

- 🌍 **Interface multilingue automatique**  
  Traduction instantanée via `googletrans` — aucune gestion manuelle de fichiers de langue.

- 📦 **Gestion complète des paquets**  
  Installation, mise à jour, suppression, inspection des versions, tout en un clic.

- ⚡ **Interface moderne et réactive**  
  UI pensée pour la clarté, l’accessibilité et la fluidité.

- 📝 **Feedback intégré**  
  Un bouton dédié pour envoyer des retours et améliorer l’application.

- 🛠️ **Architecture professionnelle**  
  Séparation nette entre logique, UI, modules, et workflow Git propre (branches dev, release, features).

## 🚀 Installation

### Windows (recommandé)
Téléchargez l’exécutable portable depuis la section **Releases**.  
Aucune installation nécessaire : double‑cliquez et utilisez PyPack immédiatement.

### Autres plateformes
```bash
git clone https://github.com/RonGarok/PyPack.git
