# ⚠️ Informations importantes concernant le listage des paquets

Lors du listage des paquets, il peut arriver que le programme interprète mal certaines données 
en scannant son Python interne. Dans ce cas, il peut n’afficher **que `pip`**.

Si cela vous arrive, vous pouvez essayer :
- Relancer le programme
- Réinstaller le programme

## 🛠️ Si le problème persiste

Si aucune de ces deux solutions ne fonctionne, vous pouvez :
- Utiliser PyPack directement via son code source
- Installer la version **V1**
- Désactiver temporairement votre antivirus  
  ou définir PyPack comme application sûre  
  *(voir LOG V1 pour plus d’informations)*

Une mise à jour est prévue prochainement.  
En attendant, vous pouvez continuer à utiliser PyPack via son code source.
